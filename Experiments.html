PsychoPy Experiments

Project Overview

This project consists of a series of vigilance tasks designed and implemented for the MIT2 Lab at UCF using PsychoPy and Pavlovia. The experiments explore topics of extended attention and focus in different modes, leveraging automated data collection and analysis to gain insights into cognitive processes.

Skills Learned

  • Experimental design and implementation with PsychoPy
  • Online experiment deployment using Pavlovia
  • Automated data collection and processing
  • Interdisciplinary collaboration with psychology researchers

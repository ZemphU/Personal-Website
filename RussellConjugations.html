Russell Conjugations Project

Project Overview

Russell Conjugations is a web application designed to identify and reverse Russell Conjugations in text using AI. The project leverages React, NodeJS, AWS Lambda, DynamoDB, and fine-tuned ChatGPT models to analyze language and provide insights into subtle linguistic framing.

Skills Learned

  • Full Stack Web Development (React, NodeJS)
  • Serverless Architecture with AWS Lambda and DynamoDB
  • Fine-tuning and deploying AI models (OpenAI API)
  • Natural Language Processing and prompt engineering
  • Project management and end-to-end deployment
